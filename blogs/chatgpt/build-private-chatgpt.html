    # 准备

    1. GitHub账号
    2. Vercel账号
    3. Openai账号及ChatGPT的apikey
    4. 可用的域名

    目前发现可在Vercel中部署的热门GPT项目有ChatGPT-Next-Web、chatbot-ui、ChuanhuChatGPT、chatgpt-demo、BetterChatGPT、Anse、yakGPT、ChatChat，这里以最高star的ChatGPT-Next-Web (opens new window)为例。

    # 1.先将ChatGPT-Next-Webfork到自己的仓库

    # 2.注册、登录Vercel，添加GitHub账号并导入该项目，在环境变量中配置OPENAI_API_KEY

    # 3.分别在Vercel和域名服务商配置域名解析

    # 4.结束
